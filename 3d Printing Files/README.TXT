This directory contains the 3d Printing files for prototype batteries,  these are all works in progress, your mileage may vary.

Version 4.0 - First prototype version incorporating a BOOST converter and Charge Controller module.  Modules will be listed in the assembly instructions which will be released as the system is built.

Version 4.0 Notes - 3D Printed 4 revisions, 4th revision modified to fit properly and is in testing as a two half design.  This is designed to fit flattest sides down on the print bed for both top and bottom covers.
